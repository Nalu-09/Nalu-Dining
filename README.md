# Nalu Dining LP — 納品物・差し替えガイド

## ファイル構成
```
nalu-dining/
├─ index.html      … 本体(開くだけで確認できます)
├─ css/style.css   … デザイン
├─ js/script.js    … 動作(設定値もここに集約)
```

## 今すぐ確認する
`index.html` をダブルクリックしてブラウザ(Safari / Chrome)で開けば、そのまま確認できます。
すべての写真は配置済みです。万一画像が見つからない場合は「◯◯.jpgを配置してください」という
案内表示になり、レイアウトは崩れません。

## 2. Instagram URLの変更方法
`js/script.js` の冒頭にある
```js
var IG_URL = "https://www.instagram.com/nalu.yuta?...";
```
このURLを書き換えるだけで、ヘッダー・ファーストビュー・固定ボタン・最終CTA・フッターなど、サイト内すべてのInstagramリンクに一括反映されます。

## 3. 問い合わせフォームの接続・非表示
`js/script.js` の
```js
var FORM_ENDPOINT = null;
```
に送信先URL(Formspreeなど)を設定すると、フォームセクションが自動的に表示されます。
`null` のままであれば、フォームセクションごと非表示になり、レイアウトは崩れません。

## 4. 法務ページURLの設定
`js/script.js` の `LEGAL_LINKS`(privacy / terms / tokushoho)にURLを設定すると、フッターのリンクに反映されます。
未設定の間はクリックすると「準備中です」という案内が表示されます。

## 5. メタ情報・OGPの変更方法
`index.html` の `<head>` 内、`<title>` と `<meta name="description">` を編集してください。
OGP画像は `images/ogp.jpg` を差し替えるだけで反映されます。
